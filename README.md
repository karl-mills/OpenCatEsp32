# OpenCatESP32

OpenCat code running on M5Stack Core2, an ESP32 controller. 


## Configuration:

ESP32 based M5Stack Core2
M5Stack Servo2 Module
M5Stack Proto 13.2 with Core2 Gyro/Mic Module inside
